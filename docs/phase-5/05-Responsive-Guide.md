# Responsive Design Guide

## Overview
This document guides viewport layouts, breakpoints, and responsive alignment parameters.

## Breakpoint Alignments

### 1. Small Screens (Mobile - `<768px`)
- **Grid Layout**: Collapses to single-column flex layouts.
- **Header**: Compact navigation.
- **Margins**: Padding scales to `4vw` to protect content space.

### 2. Medium Screens (Tablet - `768px` to `1024px`)
- **Grid Layout**: Adapts to 2-column grids for experience cards and skills.
- **Timeline**: Side-by-side timelines merge to single vertical layout stacks.

### 3. Large Screens (Desktop - `>1024px`)
- **Grid Layout**: Full multi-column layout with fixed spacing.
- **Margins**: Restores full `8vw` padding margins.
