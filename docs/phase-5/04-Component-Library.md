# Component Library Specification

## Overview
This document specifies the interactive UI components used across the redesigned public web portfolio.

## Component Inventory

### 1. Interactive Buttons
- **Primary CTA**: White background, bold tracking, with custom hover color shifts.
- **Secondary CTA**: Glass container with thin borders and micro-reveals.

### 2. Form Input Fields
- Floating label structure. Active labels scale down and transform to accent colors.
- Custom validation borders showing green/red states.

### 3. Timeline Nodes
- Vertical grid line with rounded accent borders. Nodes light up on section entry.

### 4. Code Display Cards
- High-fidelity IDE theme with status dots, file labels, and dark-code containers.
