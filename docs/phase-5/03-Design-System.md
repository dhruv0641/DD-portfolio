# Design System Framework

## Overview
This document defines the core tokens, colors, layout grids, and typography of the portfolio design system.

## Design Tokens

### 1. Colors
- **Background**: `#030303` (base canvas)
- **Surface**: `#0a0a0c` (card base)
- **Surface Hover**: `#121216` (card hover state)
- **Primary Text**: `#fafafa`
- **Muted Text**: `#a0a0a5`
- **Dim Text**: `#5c5c64`
- **Accent Blue**: `#2A56DE`

### 2. Spacing & Grids
- **Margins**: `8vw` page margins.
- **Grids**: 4-column overlay system with `1px` borders.
- **Borders**: `rgba(255, 255, 255, 0.025)` for subtle separation.

### 3. Borders & Radius
- **Border Radius SM**: `6px` (badges, tags)
- **Border Radius MD**: `12px` (input text fields, dropdowns)
- **Border Radius LG**: `20px` (showcase cards, diagrams)
