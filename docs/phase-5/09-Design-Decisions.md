# UX & Creative Design Decisions

## Overview
This document logs the creative design rationale behind the public web redesign.

## Rationale Log

### 1. Minimalist Aesthetic
- The layout focuses on high density grid systems rather than distracting colorful gradients. This mirrors Vercel/Apple style to highlight the author's code structure and professionalism.

### 2. Unified Timelines
- Merging experience and academic records into parallel side-by-side vertical columns makes scanning career achievements faster.

### 3. High Fidelity Diagrams
- The System Architecture diagram for case studies is styled as a detailed technical SVG component, immediately signaling engineering credibility.
