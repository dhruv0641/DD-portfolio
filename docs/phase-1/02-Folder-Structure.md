## 2. Directory Alignment Analysis

- **`src/types/`**: Houses all shared data models to ensure type safety across the frontend and backend.
- **`src/lib/`**: Contains core configurations and helpers (auth, constants, and utilities).
- **`backup_static/`**: Deleted to prevent cluttering the codebase.
